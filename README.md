# lescone
Le Scone um site feito em Sass (scss)

Projeto com o foco no Sass, usando como pôs processador, facilitando a codificação e a criação do grid. Gerando de forma automática todo o grid e mixins usandos, nas cores, fontes e demais função que o sass proporciona para seu css final.

Esse projeto é a conclusão do curso CSS com SASS realizado na Origamid.
